## 二、各厂商深度分析

### 1. AWS — 定价策略的行业标杆

**核心计费哲学**："像水电一样按用量计费"

**四大定价支柱：**

| 模式 | 说明 | 折扣幅度 |
|------|------|---------|
| **Pay-as-you-go** | 按实际用量付费，无需长期合同 | 基准价 |
| **Savings Plans** | 承诺 $/小时用量 1-3年 | 最高 **72%** off |
| **Volume Discounts** | S3、EC2 数据传输梯度定价，用得越多单价越低 | 随用量递增 |
| **Private Pricing** | 大客户专属承诺折扣，覆盖200+服务 | 定制化 |

**独特优势：**
- **Savings Plans** 是业界最灵活的承诺折扣——不绑定实例族、Region、OS，自动适配
- **数据传输流入免费**，只有出站收费
- 三级定价透明度：On-Demand → Savings Plans → Private Pricing
- Cost Explorer 提供 Savings Plans 智能推荐

---

### 2. Azure — 混合云与企业生态的定价专家

**核心计费哲学**："在每个阶段为你的云投资获得最佳价值"

**五大定价模式：**

| 模式 | 说明 | 折扣幅度 |
|------|------|---------|
| **Pay-as-you-go** | 按消耗付费，无预付 | 基准价 |
| **Reservations** | 预留资源 1-3年 | 最高 **72%** off |
| **Savings Plan for Compute** | 灵活的计算节省计划 | 最高 **65%** off |
| **Hybrid Benefit** | 复用已有 Windows Server/SQL Server 许可证 | 最高 **85%** off（组合时） |
| **Spot VMs** | 低价可抢占计算 | 深度折扣 |
| **Dev/Test Pricing** | 开发测试环境专属折扣 | 显著折扣 |

**独特优势：**
- **Azure Hybrid Benefit** — 独一无二，直接复用企业已有微软许可证
- **Microsoft Cost Management + Copilot** — AI 驱动的 FinOps 体验
- Azure Migrate 工具免费评估迁移成本
- 与 Microsoft 365、Dynamics 365 生态无缝集成计费
- 支持 Enterprise Agreement、MCSA 等多种合同模式

### 3. GCP — 技术创新驱动的定价差异化

> ⚠️ GCP 官网在此次调研中多次超时，以下基于模型内置知识整理。

**核心计费哲学**："为客户省钱就是为自己赢得生意"

| 模式 | 说明 | 折扣幅度 |
|------|------|---------|
| **Pay-as-you-go** | 按秒计费，最少1分钟 | 基准价 |
| **Committed Use Discounts (CUDs)** | 1-3年承诺用量 | 最高 **57%** off |
| **Sustained Use Discounts** | 自动折扣：实例运行时长越长自动降价 | 最高 **30%** off |
| **Preemptible/Spot VMs** | 抢占式实例 | 最高 **60-91%** off |
| **Free Tier** | Always Free + 90天$300试用 | 免费 |

**独特优势：**
- **Sustained Use Discounts** — 业界唯一，无需任何承诺，运行时间够长自动打折。这是GCP最创新的定价特色
- **按秒计费**（2013年率先推出），倒逼 AWS/Azure 跟进
- **Preemptible VMs** 折扣最深（最高91%）
- Preemptible VMs 固定价格（非竞价），24小时自动回收，可预测
- BigQuery 等 Serverless 产品按扫描数据量计费，模型独特
- Committed Use Discounts 可跨 Machine Type Family 共享

### 4. 阿里云 — 中国市场定价灵活度之王

**核心计费哲学**："灵活的计费选项满足一切需求"

| 模式 | 说明 | 特点 |
|------|------|------|
| **Pay-As-You-Go** | 按实际用量付费，无预付 | 弹性伸缩 |
| **Subscription（包年包月）** | 预付锁定低价 | 日常稳态业务 |
| **Resource Packages（资源包）** | 预购资源用量包 | 更灵活的折扣 |
| **Savings Plans** | ECS节省计划 | 承诺用量换折扣 |
| **Reserved Instances** | ECS预留实例 | 1-3年锁价 |
| **Storage Capacity Unit (SCU)** | 存储容量单位包 | 跨存储产品抵扣 |
| **Data Transfer Plans** | 数据传输流量包 | 降低带宽成本 |

**独特优势：**
- **计费模式组合最丰富** — 同时支持 Pay-as-you-go、Subscription、Resource Package、Savings Plan、Reserved Instance、SCU
- **资源包（Resource Package）** 概念独树一帜，跨多个产品预购抵扣
- **企业级客户可获延长账期**（完成企业认证后）
- 支持**银行转账**且免手续费
- 四级支持计划：Basic(免费) → Developer($20/月) → Business($100+/月) → Enterprise($8000+/月)
- 全球定价差异化：国际站与中国站价格体系分离

### 5. 腾讯云 — 激进的时长折扣 + 三级阶梯价

**核心计费哲学**："用得越久越便宜，用得越多越便宜"

**包年包月时长折扣（极具竞争力）：**

| 时长 | A类(S5) | B类(S9/SA5等) | C类(境外) | 通用 |
|------|---------|---------------|-----------|------|
| 1月 | 原价 | 原价 | 原价 | 原价 |
| 6-11月 | 73折 | 88折 | 68折 | 88折 |
| 1年 | 64折 | 64折 | 64折 | 83折 |
| 2年 | 54折 | 54折 | 54折 | 7折 |
| 3年 | 39折 | 39折 | 5折 | 5折 |
| 4年 | 33折 | 35折 | 5折 | 5折 |
| 5年 | **27折** | **3折** | 5折 | 5折 |

**按量计费三级阶梯价：**

| 阶梯 | 价格比例 | 说明 |
|------|---------|------|
| 第一阶梯 | 100% | 新购/变配后初始阶段 |
| 第二阶梯 | **50%** | 持续使用进入 |
| 第三阶梯 | **34%** | 进一步持续使用 |

**独特优势：**
- **5年承诺可打27折/3折** — 所有厂商中最激进的长期折扣
- **三级阶梯按量降价** — 即使不承诺，持续使用也自动降一半以上
- **按秒计费、按小时结算**
- **关机不收费** — 按量实例关机后不收CPU/内存费用
- **境外地域按量享78折**
- 变更配置后阶梯从头计，有一定约束力
- 定价策略极度细分：按实例族+地域分了A/B/C/通用四类折扣

### 6. 华为云 — 平衡策略 + 创新的竞价双模式

**核心计费哲学**："简单、透明公平、可预期的价格"

| 模式 | 说明 | 特点 |
|------|------|------|
| **按需计费** | 先使用后付费，灵活调整 | 即用即付 |
| **包年包月** | 预付费，周期越长折扣越大 | 长期稳定 |
| **阶梯计价** | 用量越大单价越低 | 大规模成本优化 |
| **竞价计费(竞价模式)** | 设价格上限，低于市场价可被回收 | 折扣价 |
| **竞价计费(竞享模式)** | 超低价格，设保障周期(最小时间单位)，到期可回收 | 超低折扣 |

**独特优势：**
- **竞价双模式** — 业界独有的「竞价模式」+「竞享模式」双层设计
  - 竞价模式：用户设最高价，供需变化时可能回收
  - 竞享模式：有保障周期(最小运行时间)，到期后可能回收，价格更低
- **竞享实例适用场景清晰定位**：图像渲染、无状态Web、基因测序、CI/CD测试等
- 明确**不适用场景**指引（长时间作业/稳定性要求极高），避免客户踩坑
- 成本中心提供异常成本检测和根因分析
- 预算管理+超预算自动提醒
- 全旅程成本管控：成本分析→预算管理→异常检测→成本优化

## 四、选择建议

### 按场景推荐

| 场景 | 首选 | 次选 | 理由 |
|------|------|------|------|
| **全球化多Region部署** | AWS | Azure | 基础设施最广，Savings Plans最灵活 |
| **微软技术栈企业** | Azure | — | Hybrid Benefit节省85% |
| **大数据/AI/ML工作负载** | GCP | AWS | Sustained Use + Preemptible最适合批处理 |
| **中国市场企业** | 阿里云 | 腾讯云 | 阿里云产品矩阵最全,腾讯云5年承诺最深 |
| **纯价格敏感/预算有限** | 腾讯云 | GCP | 腾讯云5年3折+三级阶梯极省；GCP自动折扣省心 |
| **微软许可已有大量投入** | Azure | — | Hybrid Benefit 是决定性因素 |
| **混合云/私有云场景** | 华为云 | Azure | 华为混合云经验，Azure Stack |
| **无状态/批处理/CI/CD** | GCP | 华为云 | Preemptible 91%折扣或华为竞享模式 |
| **创业/中小企业起步** | GCP | AWS | $300试用+永久免费层 |

### 核心选择逻辑

1. **已有微软 EA/License → 无脑 Azure**，Hybrid Benefit 是最大差异化优势
2. **需要自动省钱不想管 → GCP**，Sustained Use Discounts 自动生效
3. **需要最大灵活性 → AWS**，Savings Plans 不绑Instance Family
4. **中国市场 → 阿里云/腾讯云二选一**，前者生态全、后者折扣狠
5. **批处理/非关键工作负载 → GCP Preemptible 或 华为云竞享**
6. **3-5年长期锁定 → 腾讯云最便宜**，5年可打27折

## 五、趋势观察

1. **承诺折扣从"预留实例"走向"节省计划"** — AWS 率先用 Savings Plans 替代 Reserved Instances 的灵活性限制，Azure 跟进，阿里云也已推出
2. **FinOps AI化** — Azure Copilot 和 GCP Recommender 用 AI/ML 来推荐优化，成本管理正从"被动看账单"走向"AI主动优化"
3. **竞价实例双层化** — 华为云的竞价+竞享双模式可能引领新方向
4. **中国市场定价战白热化** — 腾讯云5年3折的激进策略、阿里云多模式组合，国内比国际价格更低
5. **Serverless 按调用计费** — 各厂商都在推 Serverless 产品，计费粒度从"实例/小时"走向"调用次数/GB-秒"
6. **License自带（BYOL）成为大厂竞争焦点** — Azure Hybrid Benefit 的成功可能倒逼 AWS/GCP 推出类似政策
